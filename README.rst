Description
===========

Unpack files quarantined with McAfee.

Documentation
=============

::

    Usage: unbup [-h] FILE...

    Options:
        -h, --help      Print this help and exit

    Arguments:
        FILE            Files to recover

Dependencies
============

7z - package p7zip on Archlinux
